# Moderation Model

XanaNode does not require one universal moderation authority.

Each substrate controls its own moderation rules.

Federation allows independently moderated substrates to be analyzed together while preserving their local decisions.

A substrate may choose to:

  • approve nodes
  • reject nodes
  • deprecate claims
  • preserve disputed claims
  • hide low-confidence relationships
  • publish merge reports

Moderation actions should themselves be traceable whenever possible.
